{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} ## Why Porter?

In the cloud-hosted Kubernetes cluster, the cloud providers usually provide the Load-Balancer to assign IPs and bring traffic into Kubernetes cluster. However, Kubernetes does not provide a load-balancer for bare metal cluster.

Fortunately, Porter allows you to create Kubernetes services of type LoadBalancer in bare metal cluster, which makes you enjoy the consistent experience with the cloud. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-lightbulb" title="New Feature Request" %}} If you have any new ideas or suggestions, please submit a [proposal](https://github.com/kubesphere/community/blob/master/contribution/design-proposal-template.md). {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" url="https://github.com/kubesphere/community" %}} Find something to work on from [GitHub Issue](https://github.com/kubesphere/porter/issues).
